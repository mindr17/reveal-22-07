Requirements:

Installation:


	npm install pm2 -g
					

	pm2 -v

Basic usage:


	pm2 start index.js
					

	pm2 delete all

Interface:

Access interface:


	pm2 list
or 

	pm2 l

Restart:


	pm2 restart all
or hot reload without stopping 

	pm2 reload all

More:


	pm2 stop all
					

	pm2 delete all

Single process:


	pm2 stop "process-id"
Example: 

	pm2 stop 0

Handle OS restarts:


	pm2 save
Manual, optionally: 

	pm2 resurrect
					

	pm2 startup

Check logs:


	pm2 log

Extend:


	pm2 install "module-name"
					

	pm2 uninstall "module-name"
Example: 

	pm2 install pm2-mongodb

Pros

  1. Easy to install
  2. Easy to use
  3. Big community
  4. Active support and development
  5. Supports python and other scripts

Cons

  1. Need nodejs installed
  2. Doesn't support all possible types of scripts
